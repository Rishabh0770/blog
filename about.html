About

BlogWebApp was created using HTML5, CSS3, and JavaScript. It allows users to create, manage and view blog posts. Blog data is stored temporarily in-memory, perfect for learning and testing.

*Note: Posts will not persist between sessions as no database was used in this version of the application.*
