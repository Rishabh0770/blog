About

BlogWebApp was created using HTML5, CSS3, Bootstrap 5, and vanilla JavaScript to ensure a responsive, user-friendly experience. On the backend, it uses PHP and MySQL to store blog data securely in a database, moving beyond localStorage to proper data persistence.

*Note: All blog posts are stored securely in a MySQL database, allowing them to persist across sessions and visits.*
